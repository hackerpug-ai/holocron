# Screenshots Directory

This directory contains screenshots for the Holocron application.

## Adding Screenshots

To add actual screenshots:

1. Run the app on your preferred platform (iOS, Android, or Web)
2. Take screenshots of key screens and features
3. Place them in this directory with descriptive names

## Required Screenshots

The README.md references the following screenshots (placeholders):

1. **chat-interface.png** - Main chat interface showing conversation
2. **search-results.png** - Search functionality showing document results
3. **research-workflow.png** - Deep research workflow in progress
4. **document-management.png** - Document list/management screen

## Screenshot Guidelines

- Use high-resolution images (minimum 2x for Retina displays)
- Show realistic data/content when possible
- Ensure the UI is clearly visible
- Consider light/dark mode variations
- Name files descriptively (e.g., `chat-interface-dark.png`)

## Tools for Taking Screenshots

### iOS Simulator
- Press `Cmd + S` or `File > Save Screen`
- Saves to Desktop by default

### Android Emulator
- Press `Cmd + Shift + S` (Mac) or `Ctrl + S` (Windows/Linux)
- Or use Android Studio's built-in screen capture

### Web (Expo Go)
- Use your browser's developer tools or standard screenshot tools

### Physical Devices
- iOS: Power + Home button (or Power + Volume Up on newer devices)
- Android: Power + Volume Down
