## Draft Description

Create user preference analysis in convex/feedback/analysis.ts.
Create `getUserPreferences` internalQuery: takes userId, returns preference stats.
Query feedItems by_user view, filter for userFeedback != null.
Extract positiveSources: count frequency of sources with userFeedback='up'.
Extract negativeSources: count frequency of sources with userFeedback='down'.
Return object with positiveSources (Record<string, number>), negativeSources (Record<string, number>), totalFeedback (number).
Use 30-day window: filter userFeedbackAt > Date.now() - 30*24*60*60*1000.
Count frequency with reduce: `{ [source]: count }`.
Handle no feedback case: return empty objects, totalFeedback=0.
Export from convex/feedback/index.ts.
