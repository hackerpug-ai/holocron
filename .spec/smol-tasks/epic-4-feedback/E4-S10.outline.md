## Draft Description

Add `resetPersonalization` mutation to convex/feedback/mutations.ts.
Use getAuthUserId to get current user.
Query all feedItems with userFeedback != null for this user.
Use by_viewed index with filter.
Clear userFeedback and userFeedbackAt fields on each item with ctx.db.patch().
Use Promise.all for batch updates.
Return { deleted: number } with count of cleared items.
Handle no feedback case: return { deleted: 0 }.
Export from convex/feedback/index.ts.
Use in FeedbackHistoryScreen "Clear All" action.
