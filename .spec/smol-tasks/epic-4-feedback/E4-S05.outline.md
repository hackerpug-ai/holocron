## Draft Description

Create feedback queries in convex/feedback/queries.ts.
Create `getHistory` query: returns all feedItems with userFeedback for current user, ordered by userFeedbackAt desc.
Use `by_viewed` index on feedItems table, filter for userFeedback != null.
Create `getBulkByFindings` query: takes array of feedItemIds, returns map of { feedItemId: userFeedback }.
Use v.array(v.id('feedItems')) for input.
Return object with feedItemId as key, userFeedback ('up'|'down') as value.
Handle empty input array (return empty object).
Add exports to convex/feedback/index.ts.
Use getAuthUserId, return [] if not authenticated.
