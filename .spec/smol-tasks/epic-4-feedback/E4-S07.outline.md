## Draft Description

Create feedback-aware scoring in convex/feedback/scoring.ts.
Create `calculateUserMultiplier` function: takes feedItem, UserPreferences, returns number (0.5 to 1.5).
Check feedItem.source against positiveSources: add 0.1 per feedback, max 5 = +0.5 multiplier.
Check feedItem.source against negativeSources: subtract 0.1 per feedback, max 5 = -0.5 multiplier.
Clamp final value between 0.5 and 1.5.
Create `calculatePersonalizationStrength` function: takes totalFeedback count, returns number (0.1 to 0.5).
< 5 feedback: return 0.1 (minimal personalization).
5-20 feedback: linear ramp 0.1 + (count - 5) * 0.027.
> 20 feedback: return 0.5 (max personalization).
Export both functions from convex/feedback/index.ts.
Use for feed ranking in report generation.
