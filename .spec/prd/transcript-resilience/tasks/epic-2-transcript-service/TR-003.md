## CRITICAL CONSTRAINTS

### MUST

- Use YouTube Data API v3 caption endpoint (official, ToS-compliant)
- Read existing `convex/creators/internal.ts` for YouTube API patterns
- Store full transcript in Convex file storage (not database)
- Extract previewText (first 500 chars) for search/display
- Calculate wordCount from transcript content
- Handle 404 errors (private/deleted videos) gracefully
- Handle missing captions (return empty result, don't fail)

### NEVER

- Scrape YouTube HTML (violates ToS)
- Store full transcript text in database field (use file storage)
- Throw errors for private/deleted videos (mark as failed with message)
- Use unofficial YouTube APIs or third-party caption scrapers

### STRICTLY

- Follow existing YouTube API patterns from `convex/creators/internal.ts:lookupYouTubeChannel`
- Use YOUTUBE_API_KEY environment variable (already configured)
- Fetch caption content via YouTube API, then download transcript text
- Store transcript with metadata: transcriptSource="youtube_api", transcriptType="api"
- Handle API rate limits (100 quota/day, should be sufficient for MVP)

## SPECIFICATION

export const lookupYouTubeChannel = internalAction({
  args: { handle: v.string() },
  handler: async (_, args) => {
    const apiKey = process.env.YOUTUBE_API_KEY;
    if (!apiKey) {
      console.warn("YOUTUBE_API_KEY not set, returning unverified");
      return { verified: false, error: "YouTube API key not configured" };
    }

    try {
      const response = await fetch(
        `https://www.googleapis.com/youtube/v3/channels?part=snippet,statistics&forHandle=${args.handle}&key=${apiKey}`
      );

      if (!response.ok) {
        if (response.status === 404) {
          return { verified: false, error: "Channel not found" };
        }
        throw new Error(`YouTube API error: ${response.status}`);
      }

      const data = await response.json();
      // Process response...
    } catch (error) {
      console.error("YouTube lookup error:", error);
      return { verified: false, error: error instanceof Error ? error.message : "Unknown error" };
    }
  },
});

Apply this pattern for fetchYouTubeTranscript:

```typescript
export const fetchYouTubeTranscript = internalAction({
  args: {
    contentId: v.string(), // YouTube video ID
  },
  handler: async (ctx, args) => {
    const apiKey = process.env.YOUTUBE_API_KEY;
    if (!apiKey) {
      return { hasCaptions: false, error: "YouTube API key not configured" };
    }

    try {
      // Step 1: List captions for video
      const captionsResponse = await fetch(
        `https://www.googleapis.com/youtube/v3/captions?part=snippet&videoId=${args.contentId}&key=${apiKey}`
      );

      if (!captionsResponse.ok) {
        if (captionsResponse.status === 404) {
          return { hasCaptions: false, error: "Video not found" };
        }
        throw new Error(`YouTube API error: ${captionsResponse.status}`);
      }

      const captionsData = await captionsResponse.json();

      if (!captionsData.items || captionsData.items.length === 0) {
        return { hasCaptions: false, transcript: null };
      }

      // Step 2: Download first available caption track
      const captionTrack = captionsData.items[0];
      const downloadUrl = `https://www.googleapis.com/youtube/v3/captions/${captionTrack.id}?key=${apiKey}`;
      const transcriptResponse = await fetch(downloadUrl);

      if (!transcriptResponse.ok) {
        throw new Error(`Failed to download caption: ${transcriptResponse.status}`);
      }

      const transcriptText = await transcriptResponse.text();

      // Step 3: Store in file storage
      const storageId = await ctx.storage.store(transcriptText);

      // Step 4: Extract metadata
      const previewText = transcriptText.slice(0, 500);
      const wordCount = transcriptText.split(/\s+/).length;

      return {
        hasCaptions: true,
        transcript: {
          contentId: args.contentId,
          sourceUrl: `https://www.youtube.com/watch?v=${args.contentId}`,
          transcriptType: "api",
          transcriptSource: "youtube_api",
          storageId,
          previewText,
          wordCount,
          generatedAt: Date.now(),
        },
      };
    } catch (error) {
      console.error("YouTube transcript fetch error:", error);
      return {
        hasCaptions: false,
        error: error instanceof Error ? error.message : "Unknown error",
      };
    }
  },
});
```
