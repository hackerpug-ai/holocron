# CQ-001: Audit and Categorize All Tests

> Task ID: CQ-001
> Type: AUDIT
> Priority: P1
> Estimate: 30 minutes
> Assignee: test-runner

## CRITICAL CONSTRAINTS

### MUST
- Find all test files in the repository
- Categorize each test as behavioral, existence-check, or broken
- Count `expect(fn).toBeDefined()` patterns
- Document the audit results

### NEVER

**Objective:** Create a complete inventory of all tests and categorize them by quality.
