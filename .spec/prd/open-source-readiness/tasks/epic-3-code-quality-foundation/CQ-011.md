# CQ-011: Clean Up Committed Artifacts (.bak Files)

> Task ID: CQ-011
> Type: CLEANUP
> Priority: P1
> Estimate: 15 minutes
> Assignee: backend-implementer

## CRITICAL CONSTRAINTS

### MUST
- Remove `.bak` files with `git rm`
- Target files: `components/subscriptions/SubscriptionFeedScreen.tsx.bak`, `components/subscriptions/SocialCard.tsx.bak`
- Add `*.bak` to `.gitignore`
- Verify no other `.bak` files exist

### NEVER
- Use `rm` instead of `git rm` (we want git to track the deletion)
- Leave `.bak` files in the repository

### STRICTLY
- No `.bak` files in git history after cleanup

## SPECIFICATION

**Objective:** Remove committed backup files and prevent future `.bak` files from being committed.

**Success looks like:** No `.bak` files in the repository and `*.bak` is in `.gitignore`.
