### Finding Hardcoded Border Radius
```bash
# Find borderRadius values
grep -rn "borderRadius:[ ]*[0-9]" app/ components/

# Find all border radius variants
grep -rn "border" app/ components/ | grep -i "radius"
```

### Replacement Pattern
```typescript
// ❌ BEFORE: Hardcoded border radius
const styles = StyleSheet.create({
  button: {
    borderRadius: 8,
  },
  card: {
    borderRadius: 16,
    borderTopLeftRadius: 12,
  },
})

// ✅ AFTER: Theme tokens
const { semantic } = useSemanticTheme()

const styles = useMemo(() => StyleSheet.create({
  button: {
    borderRadius: semantic.radius.md,
  },
  card: {
    borderRadius: semantic.radius.lg,
    borderTopLeftRadius: semantic.radius.md,
  },
}), [semantic])
```

### Border Radius Scale Reference
| Token | Value | Use Case |
|-------|-------|----------|
| `none` | 0px | No radius |
| `sm` | 4px | Small radius (badges, tags) |
| `md` | 8px | Medium radius (buttons) |
| `lg` | 16px | Large radius (cards) |
| `xl` | 24px | Extra large (modals) |
| `2xl` | 32px | Very large |
| `full` | 9999px | Fully rounded (pills) |

### Common Replacements
| Hardcoded | Theme Token |
|-----------|-------------|
| `0` | `semantic.radius.none` |
| `4` | `semantic.radius.sm` |
| `8` | `semantic.radius.md` |
| `12` | `semantic.radius.md` (closest) |
| `16` | `semantic.radius.lg` |
| `24` | `semantic.radius.xl` |
| `9999` | `semantic.radius.full` |
