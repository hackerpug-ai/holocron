### Finding Hardcoded Colors
```bash
# Find hex colors
grep -rn "#[0-9a-fA-F]" app/ components/

# Find rgba colors
grep -rn "rgba(" app/ components/

# Find 0x colors (React Native pattern)
grep -rn "0x[0-9a-fA-F]" app/ components/
```

### Replacement Pattern
```typescript
// ❌ BEFORE: Hardcoded colors
import { StyleSheet } from 'react-native'

const styles = StyleSheet.create({
  container: {
    backgroundColor: '#FFFFFF',
    borderColor: '#6750A4',
  },
})

// ✅ AFTER: Theme tokens
import { StyleSheet } from 'react-native'
import { useSemanticTheme } from '@/hooks/use-semantic-theme'

export const MyComponent = () => {
  const { semantic } = useSemanticTheme()

  const styles = useMemo(() => StyleSheet.create({
    container: {
      backgroundColor: semantic.color.surface.default,
      borderColor: semantic.color.primary.default,
    },
  }), [semantic])

  return <View style={styles.container} />
}

// ✅ ALTERNATIVE: Inline for dynamic values
<View style={{
  backgroundColor: semantic.color.surface.default,
  borderColor: semantic.color.primary.default,
}}>
```

### Common Replacements
| Hardcoded | Theme Token |
|-----------|-------------|
| `#FFFFFF` | `semantic.color.surface.default` |
| `#000000` | `semantic.color.onSurface.default` |
| `#6750A4` | `semantic.color.primary.default` |
| `rgba(0,0,0,0.5)` | `semantic.color.onSurface.medium` (if exists) |
