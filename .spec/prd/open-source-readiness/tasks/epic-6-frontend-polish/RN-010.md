### If Used: Correct Pattern
```typescript
import { useWindowDimensions } from 'react-native'

export function useBreakpoints() {
  const { width } = useWindowDimensions()

  return {
    isMobile: width < 768,
    isTablet: width >= 768 && width < 1024,
    isDesktop: width >= 1024,
    width,
  }
}
```

### Context
Breakpoints are typically used in web responsive design. In React Native:
- Screen sizes vary by device, not by browser window
- Use Dimensions API or useWindowDimensions hook
- Breakpoint values should be mobile-appropriate
