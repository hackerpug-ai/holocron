# RN-012: Remove or Fix Dead `useSegments` Hook

> Task ID: RN-012
> Type: REFACTOR
> Priority: P1
> Estimate: 10 minutes
> Assignee: react-native-implementer

## CRITICAL CONSTRAINTS

### MUST
- Verify if a custom `useSegments` hook exists
- Check if it's a duplicate of expo-router's built-in hook
- If duplicate wrapper: delete and use expo-router's directly

### NEVER
- Wrap built-in hooks unnecessarily

### STRICTLY
- No duplicate wrapper hooks

## SPECIFICATION

**Objective:** Remove duplicate `useSegments` hook if it's just wrapping expo-router.

**Success looks like:** Code uses expo-router's `useSegments` directly.

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | Custom `useSegments` exists | Checking implementation | Either deleted or justified | Decision documented |
| 2 | If simple wrapper | Deleting | File removed | `! ls hooks/useSegments.ts` |
| 3 | After removal | Updating imports | Uses expo-router directly | Imports from `expo-router` |
