# DS-012: Verify Images Have alt Text

> Task ID: DS-012
> Type: VERIFY
> Priority: P1
> Estimate: 20 minutes
> Assignee: react-native-reviewer

## CRITICAL CONSTRAINTS

### MUST
- Verify all images have `alt` text (web) or `accessibilityLabel` (React Native)
- Check for decorative images with empty alt
- Ensure alt text is descriptive

### NEVER
- Leave images without alt text

### STRICTLY
- All images have alt text

## SPECIFICATION

**Objective:** Verify that all images have appropriate alt text for accessibility.

**Success looks like:** Every image has descriptive alt text or is marked as decorative.

### Verification Checklist

1. **Web Images**
   - [ ] `<img>` tags have `alt` attribute
   - [ ] Alt text is descriptive
   - [ ] Decorative images have `alt=""`

2. **React Native Images**
   - [ ] `<Image>` components have `accessibilityLabel`
   - [ ] Labels are descriptive

3. **Icon Buttons**
   - [ ] Icon buttons have `aria-label` or `accessibilityLabel`

### Correct Pattern
```typescript
// ✅ GOOD: Web with descriptive alt
<img
  src="/logo.png"
  alt="Company logo"
/>

// ✅ GOOD: Decorative image
<img
  src="/decorative-pattern.png"
  alt=""
  role="presentation"
/>

// ✅ GOOD: React Native with accessibilityLabel
<Image
  source={{ uri: '/logo.png' }}
  accessibilityLabel="Company logo"
/>

// ✅ GOOD: Icon button with label
<button aria-label="Close dialog">
  <XIcon />
</button>

// ❌ BAD: Missing alt
<img src="/logo.png" />

// ❌ BAD: Unhelpful alt
<img src="/logo.png" alt="image" />
```

### Finding Issues
```bash
# Find images without alt (web)
grep -rn "<img" components/web/ | grep -v "alt="

# Find Images without accessibilityLabel (RN)
grep -rn "<Image" app/ components/ | grep -v "accessibilityLabel"

# Find icon buttons without labels
grep -rn "aria-label\|accessibilityLabel" components/ | grep -v "Icon"
```

## CODING STANDARDS

- **accessibility**: All images need alt text
- **descriptive**: Alt should describe the image content

## DEPENDENCIES

No dependencies.

## NOTES

- Alt text is critical for screen reader users
- Decorative images should have empty alt: `alt=""`
- Describe what's in the image, not just "image"
- For icons, describe the action, not the appearance
- Example: "Search" not "Magnifying glass icon"
