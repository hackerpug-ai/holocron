## DESIGN

### References
- PRD Section 2.9, AC-54
- Target: `convex/schema.ts` lines 445, 491

### Code Pattern
```typescript
// BEFORE — No validation at all
toolArgs: v.any(),

// AFTER — Enforces it's an object with string keys
toolArgs: v.record(v.string(), v.any()),

// BETTER — If you know the structure:
toolArgs: v.object({
  query: v.optional(v.string()),
  limit: v.optional(v.number()),
  // ... other known fields
}),

// ALTERNATIVE — If keys are dynamic but values are typed:
toolArgs: v.record(v.string(), v.union(
  v.string(),
  v.number(),
  v.boolean(),
  v.array(v.any()),
)),
```
