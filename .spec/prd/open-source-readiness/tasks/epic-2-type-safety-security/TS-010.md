# TS-010: Fix `searchResearch` Tool Hardcoded Status

> Task ID: TS-010
> Type: BUGFIX
> Priority: P1
> Estimate: 15 minutes
> Assignee: backend-implementer

## CRITICAL CONSTRAINTS

### MUST
- Replace hardcoded `status: "completed"` in `searchResearch` tool
- Use actual session status from the database
- Ensure search works for all session statuses

### NEVER
- Hardcode status values in queries
- Filter out sessions that aren't "completed" without reason

### STRICTLY
- Query should reflect actual session state

**Objective:** Fix hardcoded status filter in the `searchResearch` MCP tool.

**Success looks like:** The `searchResearch` tool uses the actual session status, not a hardcoded value.
