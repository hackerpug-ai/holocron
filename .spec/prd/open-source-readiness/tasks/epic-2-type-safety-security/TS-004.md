## DESIGN

### References
- PRD Section 2.2, AC-13

### Code Pattern
```typescript
// Before
cardData: v.any(),

// After - if it's a record of string keys to arbitrary values
cardData: v.record(v.string(), v.any()),

// After - if it has a known structure
cardData: v.object({
  title: v.string(),
  description: v.string(),
  metadata: v.optional(v.record(v.string(), v.any())),
}),

// Before
toolArgs: v.any(),

// After
toolArgs: v.record(v.string(), v.any()), // At minimum enforce object shape
```

## CODING STANDARDS

- **schema**: Use the most specific validator available
- **convex**: Prefer `v.record()` and `v.object()` over `v.any()`
