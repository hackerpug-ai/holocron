### MCP Section Format

```markdown
## MCP Server

Holocron includes a [Model Context Protocol (MCP)](https://modelcontextprotocol.io) server for integration with Claude Desktop and other MCP-compatible tools.

### Features

- **X+ tools** for research, document management, and subscription tracking
- Direct Convex backend integration
- Type-safe tool definitions

### Setup

1. Build the MCP server:
   \`\`\`bash
   cd holocron-mcp
   pnpm install
   pnpm build
   \`\`\`

2. Configure Claude Desktop (`~/Library/Application Support/Claude/claude_desktop_config.json`):

   \`\`\`json
   {
     "mcpServers": {
       "holocron": {
         "command": "node",
         "args": ["/path/to/holocron/holocron-mcp/dist/index.js"],
         "env": {
           "CONVEX_DEPLOYMENT": "your-deployment",
           "CONVEX_ADMIN_KEY": "your-admin-key"
         }
       }
     }
   }
   \`\`\`

3. Restart Claude Desktop

See [`holocron-mcp/README.md`](holocron-mcp/README.md) for full documentation.
```
