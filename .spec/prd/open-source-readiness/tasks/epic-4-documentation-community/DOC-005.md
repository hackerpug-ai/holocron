> Task ID: DOC-005
> Type: DOCUMENTATION
> Priority: P1
> Estimate: 30 minutes
> Assignee: documentation-writer

## CRITICAL CONSTRAINTS

### MUST
- Add screenshots or demo GIF showing the mobile app
- Include at least 3 screenshots (home, research, chat)
- Add images to `docs/screenshots/` or `assets/`
- Embed in README with proper markdown

### NEVER
- Add broken image links
- Use placeholder images

### STRICTLY
- README must include visual app preview

## SPECIFICATION

**Objective:** Add screenshots or a demo GIF to the README showing the mobile app in action.

**Success looks like:** README includes embedded images showing the app's key screens.

## CODING STANDARDS

- **presentation**: Show key features, not every screen
- **quality**: Use clear, high-resolution images
- **organization**: Keep screenshots in dedicated directory

## DEPENDENCIES

No dependencies.

## NOTES

- Take screenshots from the actual running app
- Use iOS Simulator or Android Emulator
- Key screens to capture:
  - Home/dashboard
  - Research session in progress
  - Chat interface
  - Voice recording (unique feature)
- Consider a demo GIF instead of static screenshots
- For GIF, use a tool like `gifski` or online converter
- Keep image files under 1MB each if possible
- Use PNG for screenshots, GIF for animations
- Add alt text for accessibility
