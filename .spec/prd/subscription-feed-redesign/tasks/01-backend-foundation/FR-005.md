EXAMPLES
--------------------------------------------------------------------------------

**Follow this exact pattern from convex/subscriptions/index.ts:**

```typescript
/**
 * Index file for subscriptions module
 * Exports all queries, mutations, and actions
 */

export * from "./queries";
export * from "./mutations";
export * from "./actions";
export * from "./links";

// Re-export internal module for cron jobs and internal references
export * as internal from "./internal";
```

**Why this pattern:**
- Barrel export pattern consolidates module imports
- Comment explains purpose
- Internal module exported as namespace
- All exports in one place for clean imports

**Anti-pattern (DO NOT do this):**

```typescript
// DON'T: Export non-existent files
export * from "./queries";     // Doesn't exist yet!
export * from "./mutations";   // Doesn't exist yet!

// DON'T: Mix named and namespace exports inconsistently
export * from "./queries";
export { internal } from "./internal";
```
