DESIGN
--------------------------------------------------------------------------------

**References:**
- app/(drawer)/_layout.tsx (navigation structure)
- app/chat/[conversationId].tsx (route pattern)

**Patterns to follow:**
1. **Thin wrapper**: Route file only renders screen component
2. **Default export**: Required by Expo Router
3. **testID wrapper**: E2E testing support
4. **No business logic**: Keep route file minimal

**Anti-patterns to avoid:**
1. Business logic in route file
2. State management in route file
3. Missing testID attribute
4. Named export (must be default)
