## CRITICAL CONSTRAINTS

### MUST
- All content passes quality scoring threshold (0.4+)
- Low-quality content is filtered out
- Spam and self-promotion are minimized
- Clickbait titles are downranked
- Technical depth is prioritized over superficial content

### NEVER
- Show content below quality threshold
- Promote spam or low-effort content
- Hide quality score from debugging
- Block legitimate content (false positives)

### STRICTLY
- Quality score is visible in debug mode
- Threshold is configurable (not hardcoded)
- Log filtered content for review

## SPECIFICATION

**Objective:** Ensure all content in the feed meets a minimum quality threshold to maintain user trust

**Success looks like:** User sees only high-quality content. Spam, clickbait, and low-effort posts are filtered. Quality score is visible in debug mode.

### Quality Scoring Enhancement

```typescript
// convex/whatsNew/config.ts (NEW)
export const QUALITY_CONFIG = {
  // Minimum score to be included in report (0.0 - 1.0)
  minQualityScore: 0.4,
  
  // Penalties
  clickbaitPenalty: 0.3,
  selfPromotionPenalty: 0.2,
  lowTechnicalDepthPenalty: 0.25,
  
  // Bonuses
  technicalDepthBonus: 0.15,
  authoritativeSourceBonus: 0.1,
}

// convex/whatsNew/scoring.ts
import { QUALITY_CONFIG } from './config'

interface QualityFactors {
  baseScore: number          // From LLM (0.0 - 1.0)
  clickbaitPenalty: number   // Deducted for clickbait titles
  selfPromotionPenalty: number // Deducted for self-promotion
  technicalDepthBonus: number  // Added for technical depth
}

export function calculateFinalScore(factors: QualityFactors): number {
  let score = factors.baseScore
  
  // Apply penalties
  score -= factors.clickbaitPenalty * QUALITY_CONFIG.clickbaitPenalty
  score -= factors.selfPromotionPenalty * QUALITY_CONFIG.selfPromotionPenalty
  
  // Apply bonuses
  score += factors.technicalDepthBonus * QUALITY_CONFIG.technicalDepthBonus
  
  // Clamp to 0-1
  return Math.max(0, Math.min(1, score))
}

export function passesQualityThreshold(score: number): boolean {
  return score >= QUALITY_CONFIG.minQualityScore
}

// In report synthesis
const scoredFindings = rawFindings
  .map(f => ({ ...f, qualityScore: calculateFinalScore(f.factors) }))
  .filter(f => passesQualityThreshold(f.qualityScore))
  .sort((a, b) => b.qualityScore - a.qualityScore)
```
