CODE PATTERN (Reference)

**Source**: `convex/chat/toolExecutor.ts` existing tool case switch entries

```ts
// In executeAgentTool switch
case 'find_recommendations': {
  const { query, count, location, constraints } = toolCall.args;
  const result = await ctx.runAction(internal.research.actions.findRecommendationsAction, {
    query, count, location, constraints,
  });
  return {
    messageType: 'result_card' as const,
    cardData: {
      card_type: 'recommendation_list' as const,
      items: result.items,
      sources: result.sources,
      query: result.query,
      durationMs: result.durationMs,
    },
    skipContinuation: false,
  };
}

// In agent.ts continueAfterTool
const FIND_REC_CONTINUATION_HINT =
  'The user just saw a list of recommendations. Do NOT re-render the list. ' +
  'Do NOT call another tool for the same query. ' +
  'In 1-2 sentences max, acknowledge the list and offer to save it to their KB if they want.';

if (priorTool?.name === 'find_recommendations') {
  systemMessages.unshift({ role: 'system', content: FIND_REC_CONTINUATION_HINT });
}
```

**Anti-pattern**: Do NOT set `skipContinuation=true`; do NOT call `scheduler.runAfter`; do NOT write documents.
