### Code Pattern
Source: convex/chatMessages/mutations.ts (mutation with returns validator pattern)

```typescript
export const endSession = mutation({
  args: { sessionId: v.id("voiceSessions") },
  returns: v.null(),
  handler: async (ctx, args) => {
    const session = await ctx.db.get(args.sessionId);
    if (!session) throw new Error("Session not found");
    if (session.completedAt) throw new Error("Session already ended");
    const now = Date.now();
    await ctx.db.patch(args.sessionId, {
      completedAt: now,
      totalDurationMs: now - session.startedAt,
      updatedAt: now,
    });
    return null;
  },
});
```

Do NOT create a separate voiceMessages table. Do NOT use v.any() for typed fields.
