## CRITICAL CONSTRAINTS

### MUST
- getActiveSession uses .withIndex('by_conversation') for efficient lookup
- Return null when no active session (completedAt is undefined)
- Cleanup cron times out sessions older than 10 minutes with no completedAt

### NEVER
- Use .filter() without index for session lookup
- Delete orphaned sessions — mark them completed with errorMessage

### STRICTLY
- Register cron in convex/taskCrons.ts (where crons live)
- Use internalMutation for the scheduled function
