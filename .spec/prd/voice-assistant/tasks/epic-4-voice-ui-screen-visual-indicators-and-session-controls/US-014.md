# US-014: Integrate voice UI into conversation screen

> Task ID: US-014
> Type: FEATURE
> Priority: P0
> Estimate: 60 minutes
> Assignee: react-native-ui-implementer

## CRITICAL CONSTRAINTS

### MUST
- Add VoiceMicButton to conversation screen header or input area
- Show VoiceSessionOverlay when session is active
- Wire useVoiceSession hook to UI components

### NEVER
- Break existing chat functionality — voice is additive
- Remove or hide existing chat input when voice is active
- Navigate away from conversation screen during voice session

### STRICTLY
- Use existing conversation screen layout — add voice controls, don't restructure
- Voice overlay appears above chat content, doesn't replace it

## SPECIFICATION

**Objective:** Integrate the voice session UI components into the existing conversation screen

**Success looks like:** Conversation screen has a voice button; tapping it activates the voice overlay; chat remains visible underneath

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | The conversation screen is loaded | User views the screen | VoiceMicButton is visible in the header or input area with testID 'voice-mic-button' | `pnpm vitest run --reporter=verbose 2>&1 \| grep 'voice button'` |
| 2 | User taps the voice mic button | Voice session starts successfully | VoiceSessionOverlay appears above chat content, chat still visible underneath | `pnpm vitest run --reporter=verbose 2>&1 \| grep 'overlay visible'` |
| 3 | Voice session is active | User taps stop button | Overlay dismisses, conversation screen returns to normal chat mode | `pnpm vitest run --reporter=verbose 2>&1 \| grep 'overlay dismissed'` |
| 4 | Existing chat messages in conversation | Voice session runs and records transcripts | New voice transcripts appear in the chat thread in real-time via Convex reactivity | `pnpm tsc --noEmit && pnpm vitest run` |

### Code Pattern
Source: app/(drawer)/chat/[conversationId].tsx (existing screen structure)

```typescript
import { useVoiceSession } from '@/hooks/use-voice-session';
import { VoiceMicButton, VoiceSessionOverlay } from '@/components/voice';

// Inside component:
const { state, start, stop } = useVoiceSession(conversationId);

// In JSX:
<>
  {/* Existing chat UI */}
  <VoiceMicButton state={state} onStart={start} onStop={stop} />
  {state.status !== 'idle' && (
    <VoiceSessionOverlay state={state} onRetry={start} onStop={stop} />
  )}
</>
```

### Anti-pattern (DO NOT)
Do NOT replace the chat screen with a voice-only screen. Voice augments chat.
