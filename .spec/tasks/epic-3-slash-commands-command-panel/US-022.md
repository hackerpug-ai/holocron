## CRITICAL CONSTRAINTS

### MUST
- Add "/" button to the left of ChatInput send button
- Track input value to detect when user types '/'
- Show CommandPanel when '/' is typed or button is tapped
- Pass current filter text (after '/') to CommandPanel
- Hide panel when input is cleared or loses '/' prefix

### NEVER
- Break existing send functionality
- Add keyboard navigation (deferred per deliberation)
- Hardcode panel visibility logic outside ChatInput

### STRICTLY
- Modify existing ChatInput.tsx from Epic 2
- Use state to track showCommandPanel boolean
- Extract filter from input value (text after '/')

## SPECIFICATION

**Objective:** Integrate the CommandPanel with ChatInput so that typing '/' or tapping the '/' button shows the command panel with real-time filtering.

**Success looks like:** ChatInput shows a '/' button, typing '/' triggers the panel, and the panel filters as user continues typing.

### Code Pattern

```typescript
// In ChatInput.tsx - add state and handler
import { CommandPanel } from './CommandPanel'
import type { SlashCommand } from '@/components/SlashCommandMenu'

// Add state
const [showCommandPanel, setShowCommandPanel] = useState(false)

// Track when value starts with /
const handleTextChange = (text: string) => {
  setValue(text)
  // Show panel when text starts with '/'
  setShowCommandPanel(text.startsWith('/'))
}

// Handle / button tap
const handleSlashButtonPress = () => {
  if (!value.startsWith('/')) {
    setValue('/')
  }
  setShowCommandPanel(true)
}

// Handle command selection (implemented fully in US-024)
const handleCommandSelect = (command: SlashCommand) => {
  setValue(`/${command.name} `)
  setShowCommandPanel(false)
}

// Extract filter from value
const commandFilter = value.startsWith('/') ? value : ''

// In JSX - wrap input row in relative container
<View className="relative">
  {showCommandPanel && (
    <CommandPanel
      visible={showCommandPanel}
      filter={commandFilter}
      onSelect={handleCommandSelect}
      onDismiss={() => setShowCommandPanel(false)}
    />
  )}
  <View className="flex-row items-center gap-2 px-4 py-2 border-t border-border bg-background">
    {/* / Button */}
    <Pressable
      testID="chat-input-slash-button"
      onPress={handleSlashButtonPress}
      className="w-10 h-10 rounded-full items-center justify-center bg-muted"
    >
      <Text className="font-mono text-lg text-foreground">/</Text>
    </Pressable>

    {/* Existing TextInput */}
    <TextInput
      testID="chat-input-field"
      className="flex-1 bg-muted rounded-full px-4 py-2 text-foreground"
      placeholder={placeholder}
      value={value}
      onChangeText={handleTextChange}
      onSubmitEditing={handleSend}
      editable={!disabled}
    />

    {/* Existing Send Button */}
    <Pressable
      testID="chat-input-send-button"
      onPress={handleSend}
      disabled={!canSend}
      className={cn(
        'w-10 h-10 rounded-full items-center justify-center',
        canSend ? 'bg-primary' : 'bg-muted'
      )}
    >
      <Send size={20} color={...} />
    </Pressable>
  </View>
</View>
```
