## CRITICAL CONSTRAINTS

### MUST
- Create `components/chat/ChatThread.tsx` as a scrollable list of MessageBubble components
- Create `components/chat/ChatThread.stories.tsx` with Storybook stories
- Use `FlatList` with `inverted={true}` for newest-at-bottom rendering
- Implement auto-scroll to bottom when new messages are added (via `ref.scrollToEnd()`)
- Support infinite scroll by calling `onLoadMore` when user scrolls near the top (inverted = scroll to bottom in visual terms)
- Show a loading indicator at the top (visually) when `isLoadingMore` is true
- Show an empty state when `messages` array is empty
- Accept props: `messages: ChatMessage[]`, `onLoadMore: () => void`, `isLoadingMore: boolean`, `hasMore: boolean`
- Use semantic theme tokens for all colors and spacing
- Include `testID="chat-thread"` on the container

### NEVER
- Implement actual data fetching in this DESIGN task (that's US-016 integration)
- Use `ScrollView` instead of `FlatList` (FlatList is required for virtualization)
- Hardcode colors or spacing values

### STRICTLY
- Stories must use mock message data (not real API calls)
- Play functions must verify the component renders without errors
- The FlatList must be inverted so that the newest message appears at the bottom
- `onEndReached` must trigger `onLoadMore` when `hasMore` is true

### Code Pattern

```typescript
// components/chat/ChatThread.tsx
import { useRef, useEffect } from 'react'
import { FlatList, View, StyleSheet, ActivityIndicator } from 'react-native'
import { Text } from 'react-native-paper'
import { useSemanticTheme } from '@/hooks/use-semantic-theme'
import { MessageBubble } from './MessageBubble'

export interface ChatMessage {
  id: string
  role: 'user' | 'agent' | 'system'
  content: string
  message_type: string
  created_at: string
}

export interface ChatThreadProps {
  messages: ChatMessage[]
  onLoadMore: () => void
  isLoadingMore: boolean
  hasMore: boolean
  testID?: string
}

export function ChatThread({
  messages,
  onLoadMore,
  isLoadingMore,
  hasMore,
  testID = 'chat-thread',
}: ChatThreadProps) {
  const { semantic } = useSemanticTheme()
  const flatListRef = useRef<FlatList>(null)

  // Auto-scroll to bottom when new messages arrive
  useEffect(() => {
    if (messages.length > 0) {
      // Small delay to ensure render completes
      setTimeout(() => {
        flatListRef.current?.scrollToEnd({ animated: true })
      }, 100)
    }
  }, [messages.length])

  const handleEndReached = () => {
    if (hasMore && !isLoadingMore) {
      onLoadMore()
    }
  }

  const renderItem = ({ item }: { item: ChatMessage }) => (
    <MessageBubble
      role={item.role}
      content={item.content}
      testID={`message-${item.id}`}
    />
  )

  const renderHeader = () => {
    if (!isLoadingMore) return null
    return (
      <View style={[styles.loadingContainer, { padding: semantic.space.md }]}>
        <ActivityIndicator size="small" color={semantic.color.primary.default} />
        <Text variant="bodySmall" style={{ marginLeft: semantic.space.sm }}>
          Loading more...
        </Text>
      </View>
    )
  }

  const renderEmpty = () => (
    <View style={[styles.emptyContainer, { padding: semantic.space.xl }]}>
      <Text
        variant="bodyLarge"
        style={{ color: semantic.color.onSurface.muted, textAlign: 'center' }}
      >
        No messages yet. Start a conversation!
      </Text>
    </View>
  )

  return (
    <FlatList
      ref={flatListRef}
      data={messages}
      renderItem={renderItem}
      keyExtractor={(item) => item.id}
      inverted
      contentContainerStyle={[
        styles.contentContainer,
        messages.length === 0 && styles.emptyContentContainer,
      ]}
      ListHeaderComponent={renderHeader}
      ListEmptyComponent={renderEmpty}
      onEndReached={handleEndReached}
      onEndReachedThreshold={0.5}
      testID={testID}
      style={{ backgroundColor: semantic.color.background.default }}
    />
  )
}

const styles = StyleSheet.create({
  contentContainer: {
    paddingVertical: 8,
  },
  emptyContentContainer: {
    flexGrow: 1,
    justifyContent: 'center',
  },
  loadingContainer: {
    flexDirection: 'row',
    alignItems: 'center',
    justifyContent: 'center',
  },
  emptyContainer: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
})
```

## NOTES
- The FlatList is inverted because chat UIs typically show newest messages at the bottom. With inversion, data order (newest first from API) renders correctly.
- `onEndReached` fires when scrolling toward the "end" of the list, which visually is the top (older messages) due to inversion.
- Auto-scroll uses `scrollToEnd` which, with inversion, scrolls to the visual bottom (newest messages).
- Real data integration happens in US-016; this task uses only mock data.
