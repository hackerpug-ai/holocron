# Build chat-send Edge Function with basic agent response routing

> Task ID: US-011
> Type: feature
> Priority: P1
> Appetite Fit: medium (fits)
> Assignee: supabase-implementer
> PRD Version: 3.1.0
> Stability: semi-stable

## CRITICAL CONSTRAINTS

### MUST
- Accept POST requests with conversation_id, content, message_type in request body
- Validate that content is not empty or whitespace-only
- Verify conversation exists before inserting message
- Insert user message into chat_messages table with role='user'
- Update conversation updated_at and last_message_preview
- Generate and insert stub agent response with role='agent'
- Return user_message_id and array of agent_messages

### NEVER
- Accept GET requests (POST only)
- Insert messages without conversation validation
- Return raw database errors to client
- Use client-side Supabase module (create server-side client)

### STRICTLY
- Use shared CORS headers from _shared/cors.ts
- Follow Deno.serve() pattern from conversations function
- Type all request/response interfaces
- Truncate last_message_preview to 100 characters

## SPECIFICATION

**Objective:** Create the chat-send Edge Function that accepts user messages, persists them to the database, generates a stub agent response, and returns both the user message ID and agent response for the UI.

**Success looks like:** POST /functions/v1/chat-send accepts messages, validates input, persists to database, generates stub responses, and returns structured JSON with user_message_id and agent_messages array.

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | A valid conversation_id and non-empty content | POST is sent to /functions/v1/chat-send | Response is 200 with user_message_id (UUID) and agent_messages array with at least one message | `curl -X POST $SUPABASE_URL/functions/v1/chat-send -H 'Authorization: Bearer $ANON_KEY' -H 'Content-Type: application/json' -d '{"conversation_id":"...","content":"Hello"}'` |
| 2 | An empty content string or whitespace only | POST is sent to /functions/v1/chat-send | Response is 400 with error message 'Message content cannot be empty' | `curl -X POST ... -d '{"conversation_id":"...","content":"   "}' \| jq '.error'` |
| 3 | A non-existent conversation_id | POST is sent to /functions/v1/chat-send | Response is 404 with error message 'Conversation not found' | `curl -X POST ... -d '{"conversation_id":"00000000-0000-0000-0000-000000000000","content":"test"}'` |
| 4 | A successful message send | The response is returned | Conversation.updated_at is updated and last_message_preview contains truncated message | `SELECT updated_at, last_message_preview FROM conversations WHERE id = :id` |

### Code Pattern

```typescript
// supabase/functions/chat-send/index.ts
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import { corsHeaders } from '../_shared/cors.ts'

interface ChatSendRequest {
  conversation_id: string
  content: string
  message_type?: 'text' | 'slash_command'
}

interface ChatSendResponse {
  user_message_id: string
  agent_messages: Array<{
    id: string
    role: 'agent'
    content: string
    message_type: string
    created_at: string
  }>
}

Deno.serve(async (req: Request) => {
  if (req.method === 'OPTIONS') {
    return new Response(null, { status: 204, headers: corsHeaders })
  }

  if (req.method !== 'POST') {
    return jsonResponse({ error: 'Method not allowed' }, 405)
  }

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!
  )

  try {
    const body: ChatSendRequest = await req.json()
    
    // Validate content
    if (!body.content || body.content.trim().length === 0) {
      return jsonResponse({ error: 'Message content cannot be empty' }, 400)
    }

    // Verify conversation exists
    const { data: conversation, error: convError } = await supabase
      .from('conversations')
      .select('id')
      .eq('id', body.conversation_id)
      .single()

    if (convError || !conversation) {
      return jsonResponse({ error: 'Conversation not found' }, 404)
    }

    // Insert user message
    const { data: userMsg, error: userMsgError } = await supabase
      .from('chat_messages')
      .insert({
        conversation_id: body.conversation_id,
        role: 'user',
        content: body.content.trim(),
        message_type: body.message_type || 'text'
      })
      .select()
      .single()

    // Generate stub agent response
    const agentContent = `I received your message: "${body.content.trim().substring(0, 50)}..."`
    const { data: agentMsg } = await supabase
      .from('chat_messages')
      .insert({
        conversation_id: body.conversation_id,
        role: 'agent',
        content: agentContent,
        message_type: 'text'
      })
      .select()
      .single()

    // Update conversation
    await supabase
      .from('conversations')
      .update({
        updated_at: new Date().toISOString(),
        last_message_preview: body.content.trim().substring(0, 100)
      })
      .eq('id', body.conversation_id)

    return jsonResponse({
      user_message_id: userMsg.id,
      agent_messages: [agentMsg]
    })
  } catch (error) {
    console.error('chat-send error:', error)
    return jsonResponse({ error: 'Internal server error' }, 500)
  }
})

function jsonResponse(body: unknown, status = 200): Response {
  return new Response(JSON.stringify(body), {
    status,
    headers: { ...corsHeaders, 'Content-Type': 'application/json' }
  })
}
```
