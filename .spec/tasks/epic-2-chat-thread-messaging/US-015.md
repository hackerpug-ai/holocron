## CRITICAL CONSTRAINTS

### MUST
- Include TextInput for message composition
- Include Send button that is disabled when input is empty/whitespace
- Use NativeWind utility classes for all styling
- Use `cn()` from `@/lib/utils` for conditional class merging
- Import `Text` from `@/components/ui/text` -- never from `react-native`
- Include `testID` on every interactive element
- Co-locate a Storybook story file (`ChatInput.stories.tsx`) as a sibling
- Support `onSend(content: string)` callback
- Clear input after successful send

### NEVER
- Never hardcode hex colors -- use NativeWind semantic tokens
- Never hardcode spacing values -- use Tailwind spacing classes
- Never import `Text` from `react-native` -- always use `@/components/ui/text`
- Never add API fetching logic -- this is a design-only task
- Never allow sending empty or whitespace-only messages

### STRICTLY
- All props must be TypeScript interfaces with JSDoc comments
- Component file must be named `ChatInput.tsx` and export a named function `ChatInput`
- Send button must show disabled state visually when input is empty
- Support keyboard "return" key to send (submitBehavior)

## SPECIFICATION

**Objective:** Create a `ChatInput` component with a text input field and send button, implementing empty message validation and clear-on-send behavior.

**Success looks like:** A chat input bar in Storybook that validates input, enables/disables the send button appropriately, and calls `onSend` with trimmed content when submitted.

### Play Functions
```typescript
import { within, userEvent, expect } from '@storybook/test'

export const Interactive: Story = {
  render: () => {
    const [lastSent, setLastSent] = useState<string | null>(null)
    return (
      <View>
        <ChatInput onSend={setLastSent} />
        {lastSent && <Text>Last sent: {lastSent}</Text>}
      </View>
    )
  },
  play: async ({ canvasElement }) => {
    const canvas = within(canvasElement)
    const input = canvas.getByTestId('chat-input-field')
    const sendButton = canvas.getByTestId('chat-input-send-button')

    // Initially disabled
    await expect(sendButton).toBeDisabled()

    // Type text
    await userEvent.type(input, 'Hello world')
    await expect(sendButton).toBeEnabled()

    // Send
    await userEvent.click(sendButton)
    await expect(input).toHaveValue('')
    await expect(canvas.getByText('Last sent: Hello world')).toBeTruthy()
  },
}
```

### Code Pattern
```typescript
// components/chat/ChatInput.tsx
import { View, TextInput, Pressable } from 'react-native'
import { Text } from '@/components/ui/text'
import { cn } from '@/lib/utils'
import { useState } from 'react'
import { Send } from 'lucide-react-native'

export interface ChatInputProps {
  /** Callback when user sends a message */
  onSend?: (content: string) => void
  /** Placeholder text for input */
  placeholder?: string
  /** Whether the input is disabled */
  disabled?: boolean
  /** Default value for the input */
  defaultValue?: string
}

export function ChatInput({
  onSend,
  placeholder = 'Type a message...',
  disabled = false,
  defaultValue = '',
}: ChatInputProps) {
  const [value, setValue] = useState(defaultValue)

  const trimmedValue = value.trim()
  const canSend = trimmedValue.length > 0 && !disabled

  const handleSend = () => {
    if (canSend && onSend) {
      onSend(trimmedValue)
      setValue('')
    }
  }

  return (
    <View
      testID="chat-input"
      className="flex-row items-center gap-2 px-4 py-2 border-t border-border bg-background"
    >
      <TextInput
        testID="chat-input-field"
        className="flex-1 bg-muted rounded-full px-4 py-2 text-foreground"
        placeholder={placeholder}
        placeholderTextColor="hsl(var(--muted-foreground))"
        value={value}
        onChangeText={setValue}
        onSubmitEditing={handleSend}
        editable={!disabled}
        returnKeyType="send"
        blurOnSubmit={false}
      />
      <Pressable
        testID="chat-input-send-button"
        onPress={handleSend}
        disabled={!canSend}
        className={cn(
          'w-10 h-10 rounded-full items-center justify-center',
          canSend ? 'bg-primary' : 'bg-muted'
        )}
      >
        <Send
          size={20}
          color={canSend ? 'hsl(var(--primary-foreground))' : 'hsl(var(--muted-foreground))'}
        />
      </Pressable>
    </View>
  )
}
```
