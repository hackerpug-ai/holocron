## SPECIFICATION

**Objective:** Create session list card shown when user types `/resume` to select incomplete sessions

**Success looks like:** User sees list of incomplete sessions with topic, last iteration, coverage score, and date

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | User types `/resume` | Session list renders | Shows all incomplete sessions as selectable cards | `story renders with MultipleSessions variant` |
| 2 | Session list has sessions | Each session card displays | Shows topic, last iteration (e.g., '2/5'), coverage score, and date started | `story shows session metadata` |
| 3 | No incomplete sessions exist | Session list renders | Empty state shown with message | `story renders EmptyState variant` |
| 4 | User taps session card | onSelect callback fires | Session ID passed to callback | `story play function tests selection` |
