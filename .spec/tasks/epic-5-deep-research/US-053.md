## SPECIFICATION

**Objective:** Create detail view for completed deep research with iteration timeline and citations

**Success looks like:** User sees full synthesized report, iteration timeline with score progression, expandable iterations, and all citations

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | User taps completed result card | Detail view renders | Shows synthesized report with markdown rendering | `story renders with Default variant` |
| 2 | Detail view displays | Iteration timeline shown | Shows score progression (e.g., 2→3→4→5) with visual indicator | `story shows timeline visualization` |
| 3 | User taps iteration in timeline | Iteration expands | Shows findings and reviewer feedback for that iteration | `story play function tests expansion` |
| 4 | Detail view displays | Citations section visible | All citations accumulated across iterations listed | `story shows citations list` |
| 5 | Detail view displays | Holocron save confirmation shown | Message indicates report saved to knowledge base | `story shows save confirmation` |
