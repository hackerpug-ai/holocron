# Build /browse command handler with categories

> Task ID: US-027
> Type: feature
> Priority: P1
> Appetite Fit: medium (fits)
> Assignee: supabase-implementer
> PRD Version: 3.1.0
> Stability: evolving

## CRITICAL CONSTRAINTS

### MUST
- Handle /browse command (with and without category argument)
- Return category list card when /browse has no args
- Return article cards filtered by category when /browse <category> provided
- Sort articles by most recent first (created_at DESC)
- Limit results to 10 articles per category

### NEVER
- Implement pagination "show more" (deferred)
- Create new category listing RPC (query documents table directly)
- Return results as plain text (must use card_data)

### STRICTLY
- Use document_category enum: architecture, business, competitors, frameworks, infrastructure, libraries, patterns, platforms, security, research
- Store message_type='result_card' for all responses
- Return "category not found" card for invalid category names

## SPECIFICATION

**Objective:** Implement the /browse command handler that lists categories or filters articles by category, returning results as tappable result cards.

**Success looks like:** Submitting '/browse' shows a category list card with counts; '/browse architecture' shows article cards for that category.

### Code Pattern
```typescript
if (parsed.isCommand && parsed.command === 'browse') {
  const categoryArg = parsed.args

  if (!categoryArg) {
    // List all categories with counts
    const { data: categories } = await supabase
      .from('documents')
      .select('category')
      .not('category', 'is', null)

    const categoryCounts = categories?.reduce((acc, doc) => {
      acc[doc.category] = (acc[doc.category] || 0) + 1
      return acc
    }, {})

    agentContent = 'Browse articles by category'
    agentCardData = {
      card_type: 'category_list',
      categories: Object.entries(categoryCounts).map(([name, count]) => ({ name, count }))
    }
  } else {
    // Filter by category
    const { data: articles } = await supabase
      .from('documents')
      .select('id, title, date, research_type')
      .eq('category', categoryArg)
      .order('created_at', { ascending: false })
      .limit(10)

    if (!articles || articles.length === 0) {
      agentContent = `Category "${categoryArg}" not found`
      agentCardData = {
        card_type: 'category_not_found',
        valid_categories: ['architecture', 'business', ...]
      }
    } else {
      agentContent = `Found ${articles.length} articles in ${categoryArg}`
      agentCardData = articles.map(doc => ({
        card_type: 'article',
        title: doc.title,
        date: doc.date,
        research_type: doc.research_type,
        document_id: doc.id
      }))
    }
  }
}
```
