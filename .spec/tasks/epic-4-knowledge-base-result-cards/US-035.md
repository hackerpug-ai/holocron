# Wire drawer Articles link to ArticlesScreen

> Task ID: US-035
> Type: feature:integration
> Priority: P2
> Appetite Fit: small (fits)
> Assignee: react-native-ui-implementer
> PRD Version: 3.1.0
> Stability: evolving

## CRITICAL CONSTRAINTS

### MUST
- Add Articles route to Expo Router
- Wire drawer "Articles" link to navigate to ArticlesScreen
- Preserve drawer navigation context
- Include back button or drawer access in ArticlesScreen

### NEVER
- Break existing drawer navigation
- Create new navigation pattern (use Expo Router)

### STRICTLY
- Use Expo Router navigation
- Articles link in drawer opens ArticlesScreen
- Back button returns to chat

## SPECIFICATION

**Objective:** Connect the drawer "Articles" link to the ArticlesScreen so users can navigate to the dedicated articles browsing view.

**Success looks like:** Tapping "Articles" in drawer opens ArticlesScreen with full article list, filters, and search.

## ACCEPTANCE CRITERIA

| # | Given | When | Then | Verify |
|---|-------|------|------|--------|
| 1 | User taps "Articles" in drawer | onPress fires | Navigation to ArticlesScreen occurs | `pnpm dev` + manual inspection |
| 2 | ArticlesScreen opens | Screen renders | Back button or drawer access available | `pnpm dev` + manual inspection |
| 3 | User navigates back | Back button/tap | Returns to chat screen | `pnpm dev` + manual inspection |
