## Architecture

This app uses **Convex** as the backend:
- **Database**: Document storage, vector embeddings, chat history
- **Real-time**: Automatic updates via useQuery
- **AI Agents**: GPT-5 (lead) + GPT-5-mini (subagents) for deep research
- **Workflows**: Convex Workflow for orchestrating multi-agent research

### Key Features
- Multi-agent deep research with cost-optimized GPT-5-mini subagents
- Real-time progress tracking (no manual subscriptions needed)
- Vector search + full-text search for document retrieval
