## DESIGN
### Pattern (Command Migration)
```typescript
// brain/skills/holocron/SKILL.md

## Commands

### /holocron search <query>

Search holocron using hybrid search (vector + FTS).

**Before (Supabase):**
```typescript
const { data, error } = await supabase.rpc('hybrid_search', {
  query: args.query,
  match_count: 10,
});
```

**After (Convex):**
```typescript
const results = await client.query(api.documents.hybridSearch, {
  query: args.query,
  limit: 10,
});

return {
  results: results.map(doc => ({
    id: doc._id,
    title: doc.title,
    category: doc.category,
    snippet: doc.content.substring(0, 200),
    score: doc._score,
  })),
};
```

### /holocron list [--category <cat>] [--limit <n>]

List documents from holocron.

**After (Convex):**
```typescript
const documents = await client.query(api.documents.list, {
  category: args.category,
  limit: args.limit ?? 20,
});

return {
  documents: documents.map(doc => ({
    id: doc._id,
    title: doc.title,
    category: doc.category,
    createdAt: new Date(doc.createdAt).toISOString(),
  })),
  total: documents.length,
};
```

### /holocron get <id>

Get a specific document.

**After (Convex):**
```typescript
const doc = await client.query(api.documents.get, {
  id: args.id,
});

if (!doc) {
  throw new Error(`Document ${args.id} not found`);
}

return {
  document: {
    id: doc._id,
    title: doc.title,
    content: doc.content,
    category: doc.category,
    filePath: doc.filePath,
    fileType: doc.fileType,
    createdAt: new Date(doc.createdAt).toISOString(),
  },
};
```

### /holocron stats

Show holocron statistics.

**After (Convex):**
```typescript
const stats = await client.query(api.documents.stats, {});

return {
  total: stats.total,
  byCategory: stats.byCategory,
  recentDocuments: stats.recentDocuments.map(doc => ({
    id: doc._id,
    title: doc.title,
    createdAt: new Date(doc.createdAt).toISOString(),
  })),
};
```

### /holocron browse [--category <cat>]

Interactive document browser.

**After (Convex):**
```typescript
const documents = await client.query(api.documents.list, {
  category: args.category,
  limit: 50,
});

// Present for user selection
return {
  type: "interactive",
  documents: documents.map(doc => ({
    id: doc._id,
    title: doc.title,
    category: doc.category,
  })),
  onSelect: async (selectedId: string) => {
    const doc = await client.query(api.documents.get, { id: selectedId });
    return { document: doc };
  },
};
```
```

### Anti-pattern (DO NOT)
```typescript
// BAD: Changing command interface
/holocron find <query>  // WRONG - keep "search"

// BAD: Not handling Convex ID format
doc.id  // WRONG - Convex uses doc._id

// BAD: Not using generated API
client.query("documents", "list", {})  // WRONG - use api.documents.list

// BAD: Mixing Supabase and Convex
const supabaseResults = await supabase.from('documents')...
const convexResults = await client.query(...)
// Pick one and stick with it
```
