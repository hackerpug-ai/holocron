## DESIGN
### Pattern (ConvexHttpClient for CLI)
```typescript
// brain/skills/holocron/SKILL.md - Update tool definitions

## Tools

### holocron_search

Search holocron knowledge database.

**Parameters:**
- query (required): Search query string

**Usage:**
```typescript
import { ConvexHttpClient } from "convex/browser";
import { api } from "./convex/_generated/api";

const client = new ConvexHttpClient(process.env.CONVEX_URL!);

const results = await client.query(api.documents.hybridSearch, {
  query: args.query,
  limit: 10,
});

return {
  results: results.map(doc => ({
    id: doc._id,
    title: doc.title,
    category: doc.category,
    snippet: doc.content.substring(0, 200),
  })),
};
```

### holocron_list

List documents from holocron.

**Parameters:**
- category (optional): Filter by category
- limit (optional): Maximum results (default 20)

**Usage:**
```typescript
const documents = await client.query(api.documents.list, {
  category: args.category,
  limit: args.limit ?? 20,
});

return {
  documents: documents.map(doc => ({
    id: doc._id,
    title: doc.title,
    category: doc.category,
    createdAt: doc.createdAt,
  })),
};
```

### holocron_get

Get a specific document by ID.

**Parameters:**
- id (required): Document ID

**Usage:**
```typescript
const doc = await client.query(api.documents.get, {
  id: args.id,
});

if (!doc) {
  throw new Error(`Document ${args.id} not found`);
}

return {
  document: {
    id: doc._id,
    title: doc.title,
    content: doc.content,
    category: doc.category,
    createdAt: doc.createdAt,
  },
};
```
