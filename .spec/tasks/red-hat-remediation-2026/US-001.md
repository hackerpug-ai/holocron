## EXAMPLES

**Follow this exact pattern for drawer layout:**

```typescript
// app/(drawer)/_layout.tsx
import { Drawer } from 'expo-router/drawer'
import { DrawerContent } from '@/components/DrawerContent'

export default function Layout() {
  return (
    <Drawer
      drawerContent={(props) => <DrawerContent {...props} />}
      screenOptions={{
        headerShown: false,
        drawerPosition: 'left',
        drawerType: 'front',
      }}
    >
      <Drawer.Screen
        name="index"
        options={{
          title: 'Chat',
        }}
      />
    </Drawer>
  )
}
```

**Why this pattern:**
- Uses Expo Router's built-in drawer navigation
- Composes with existing DrawerContent component
- Follows file-based routing convention
- Provides gesture-based drawer open/close

**Anti-pattern (DO NOT do this):**

```typescript
// ❌ NO DRAWER STRUCTURE
export default function Layout() {
  return <Stack>{/* No drawer! */}</Stack>
}
```
